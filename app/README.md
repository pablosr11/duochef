# ChefAtHome App (MVP)

This is the interactive mobile app for ChefAtHome, built with **React Native** and **Expo**.

## 🚀 Quick Start
1. Ensure you have Node.js installed.
2. Install the Expo CLI: `npm install -g expo-cli`
3. Install dependencies: `npm install`
4. Start the app: `npx expo start`

## 📱 Features
- **Duolingo-style Skill Tree**: Progress through foundational cooking skills.
- **Interactive Player**: Step-by-step guidance for safe and confident cooking.
- **Content-Ready**: Pre-loaded with lessons on Safety, Boiling, and Knife Skills.

## 🛠 Tech Stack
- React Native
- Expo
- Lucide-React-Native (Icons)
- React Navigation

## 📂 Structure
- `App.js`: Main application entry and UI logic.
- `lessons/`: JSON files defining lesson steps and timers.
- `spec.md`: The product blueprint.
