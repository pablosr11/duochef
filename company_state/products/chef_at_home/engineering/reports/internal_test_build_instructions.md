# ChefAtHome Internal Test Build Instructions

## Overview
This document provides instructions for installing and testing the ChefAtHome application.

## Prerequisites
*   A device running iOS.
*   TestFlight installed on your device.

## Installation (TestFlight)
1.  Open the TestFlight app on your device.
2.  You should see ChefAtHome listed as a new app to test. If not, check your email for an invitation.
3.  Tap "Install" next to ChefAtHome.
4.  Once the app is installed, tap "Open".

**Alternative Installation Method (if TestFlight is unavailable):**
1.  Connect your iOS device to your computer.
2.  Open Xcode.
3.  Go to Window -> Devices and Simulators.
4.  Select your device.
5.  Click the "+" button under "Installed Apps".
6.  Select the ChefAtHome .ipa file (provided separately).

## Obtaining the Build
The ChefAtHome .ipa file will be provided to you by the engineering team. Please contact the engineering team if you have not received the latest build.

## Testing
1.  Navigate through Level 0 and Level 1 lessons.
2.  Verify that all interactive elements function correctly.
3.  Check that progress (XP, streaks) is saved correctly.
4.  Report any bugs or issues using the feedback mechanism.

## Feedback
Please provide feedback to the marketing team.
