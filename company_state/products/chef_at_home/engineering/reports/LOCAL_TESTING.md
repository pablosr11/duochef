# Local Testing Instructions for ChefAtHome

## Prerequisites
1. Ensure you have Node.js and npm installed.
2. Clone the repository from GitHub.
3. Navigate to the project directory.

## Running the App Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open your browser and go to `http://localhost:3000`.

## Testing the Waitlist Feature
1. Ensure the waitlist functionality is implemented.
2. Test the email submission form on the landing page.
3. Verify that submitted emails are stored correctly in the local JSON file or database.

## Safety Disclaimer
- The app provides general educational guidance, not professional culinary training.
- Users must follow local food safety guidance and agree to the terms before starting any lesson.
